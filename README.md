1. [搜狗拼音](https://pinyin.sogou.com/linux/?r=pinyin)

2. [chrome浏览器](https://www.google.com/chrome/)

4. [wps office for linux](https://www.wps.com/wps-office-for-linux)

   自己觉得比 ubuntu 自带的好用
